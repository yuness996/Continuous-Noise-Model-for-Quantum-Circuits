# Continuous-Noise-Model-for-Quantum-Circuits
Code accompanying the paper "Continuous Noise Model for Quantum Circuits", [arXiv:2604.26008](https://arxiv.org/abs/2604.26008). Includes simulations of continuous coherent and Pauli noise, entropy-matched comparisons, analytical error propagation, quantum error-correcting codes ([[5,1,3]] and [[7,1,3]]), Grover circuits, and random Clifford circuits.
